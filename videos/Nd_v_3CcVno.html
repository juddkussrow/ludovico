No online-only requirement (unlike Notion offline toggle). Obsidian model: files locally available like normal documents. Cache control for large media (previews local, full files on-demand). (2) Conflict-Free Syncing via CRDTs—traditional syncing overwrites or creates conflicts (newest wins, or reload/sync conflict dialogs). Obsidian phone vault example: two tapped notes created empty files, deleted originals on other devices via sync. CRDTs save notes as series of inputs tracking insertion location, allow both devices to work, merge when online without overwriting. Live sync streams changes between editors. Server disappearance irrelevant; resync from clients possible. (3) Flexible Linking Over Folders—folders force one place, creating duplicates (multi-place dilemma) or lost items in deep subfolders. Photos example: browsing endless scroll (no tags), can't easily retrieve meme to show friend, no personal context captured. Linking is browsable, intuitive (Wikipedia game example). Heaper approach: bottom-up organization (create first, structure emerges). Mentions (@ character) link blocks; tags define is/has relations. Unlinked view shows unsorted items (visual progress). Block = file + note + rich text context + tags/mentions (no sidecar files). Anything can link anything without duplicates. Try new structures atop old without deleting. (4) True Ownership—self-host on own hardware with Docker, or use hosted EU cloud. No vendor lock-in. Open formats insufficient (Obsidian linking only works in Obsidian; photos still need Obsidian). Self-hosting gives sync + thumbnails + privacy (AES encryption at rest, TLS in transit, not end-to-end). Decentralized option possible later (SSH + PGP, no blockchain). Strong encryption on cloud; no E2E due to regulatory complexity (Apple disabled advanced data protection in UK). Pricing: free offline version (no account), free self-hosting (up to 50GB), paid subscriptions for cloud or larger self-hosting unlock. Cross-platform (iOS, Android, Windows, Linux, Mac). Features: journaling (daily/weekly), fast media browsing, live multi-device typing, conflict-free merges, bulk organizing, lightning-fast local search, auto deduplication, device caching. Local SQLite + JSON/markdown backups (no pure CRDT trust). Roadmap: Kanban, tasks, calendar, graph, properties, databases, saved filtered views, teleprompter.
