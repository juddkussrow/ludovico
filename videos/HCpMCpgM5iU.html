Models+training data+code all released on HuggingFace/GitHub. ReflectionFlow: iterative image refinement plugin for Flux1Dev—generates multiple images in parallel via scaling noise, selects best by prompt, iteratively refines via scaling reflection (fixes missing objects, color, shape) or scaling prompt (LLM enhances prompt, rejects if misaligned). Improves accuracy for complex multi-object prompts vs single-pass generation. ReflectionFlow open-sourced on GitHub. Hunyuan 3D-2.5 (Tencent): described as best image-to-3D generator tested. Sign up via WeChat/QQ/email at platform. Front/rear/multi-view image upload or text prompt. Generates PBR maps (albedo, normal, texture), detailed clothing/accessories, 3D consistent. Example: anime girl complex dress—generated with intricate dress details, back-view guesstimated accurately. Download in multiple formats. Other user examples equally impressive (realistic, metallic textures, ribbon connectivity). Version 2.5 not yet open-sourced (online-only for now). Uni3C: generate video with dual control—camera trajectory + character motion. Input image→3D point cloud; user specifies camera path in 3D space; outputs video with cinematic camera movement. Also: input reference video (person dancing/walking), maps pose/hand/finger movements onto generated character. Combines both controls: same dance + different camera angles, or different dances + same scene. Uses Alibaba's Wan as base video generator. Code+models coming soon (GitHub). Xpeng Iron: 178cm humanoid robot, Xpeng's proprietary Turing AI chip, tested in production lines (part assembly, sorting), mass-production planned next year, est. $150k/robot, for warehouses/factories. Magi-1: open-source image-to-video by SandAI. Autoregressive architecture (not diffusion)—predicts video chunk by chunk vs whole-at-once. 24B (full) requires 8 H100s, 24B distilled+quantized requires 8 RTX 4090s, 4.5B version (not yet released, planned end-April, runs on 1 RTX 4090). Apache 2 license (commercial OK). Quality issues in tests: warping, face changes, prompt-following inferior to Wan/Kling 2.0. SkyReels V2: open-source infinite-length video (diffusion forcing variant). 1.3B requires ~15GB VRAM, 14B requires ~51GB VRAM. 28-30s video examples (swan, jellyfish) very high quality, realistic water/coral details. Tested via HuggingFace (free account, 25 credits=1 video): zombie mall prompt generated low-quality output, not as good as Wan. GitHub repo + online platform available. Dia TTS (Nari Labs): 1.6B text-to-dialogue model, multi-speaker, voice cloning from reference audio. Outperforms 11Labs/Sesame in demos (laugh quality, natural timing).
