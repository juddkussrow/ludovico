Graphify: one-time graph construction using tree-sitter (syntax parsing) + LLM (meaning extraction) + clustering (relationship inference); generates visual graph, written report (Markdown), and queryable knowledge base. Result: subsequent queries drop from ~14,000 tokens to ~a few hundred on the same question, because the graph is cached and only updates on file changes. RAG (similarity search) fails for code because chunks aren't semantically contiguous (functions call across files). Graphify builds real relationships: "this function calls that one," "this module depends on that," "this idea came from this document"—with confidence labels (extracted, inferred, ambiguous).
