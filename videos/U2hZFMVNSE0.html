Modern LLMs like GPT, Gemini, Deep Seek use identical decoder-only variants. Core process: predict next most probable word, append to sentence, repeat. Pipeline: (1) Tokenization: break text into subwords (prefixes, roots, suffixes) capturing semantic meaning while maintaining manageable vocabulary size and handling unknown/novel words. (2) Input Embedding: convert token IDs into high-dimensional vectors capturing word meaning (GPT-3 uses 12,288 numbers per vector). Words with similar meanings cluster in high-dimensional space (e.g., man/woman/boy/girl form gender/age coordinates). (3) Positional Encoding: sine/cosine waves encode word position (0,1,2,3,4) into unique fingerprints added to embeddings so model knows word order ("dog bit cat" vs. "cat bit dog" are different). (4) Masked Multi-Head Attention: core breakthrough. Query (asks "what's relevant to me?"), Key (raises hand with label), Value (provides substance if key matches). Calculate attention: dot product of Q×K matrices (similarity scores) masked to prevent looking at future words, softmax for normalized probabilities, multiply by V vectors. Attention enriches each word vector with context from all preceding words. Multiple heads (typically 8+) each learn different contextual relationships (subject-verb, pronoun reference, etc.). Concatenate and multiply by WO matrix. (5) Add & Norm: skip connection adds original input back (residual connection prevents information loss in deep networks), layer normalization scales values (mean 0, std 1). (6) Feed-Forward Neural Network: expands vectors 2-4x, then contracts back to original size. Provides extra thinking/processing time based on gathered context. (7) Add & Norm: repeat. (8) Multiple Decoder Blocks: stack blocks for progressively deeper meaning extraction. Block 1 detects basic relationships; Block 2 finds deeper meaning; Block 3+ finds even deeper relationships. (9) Final Linear Layer + Softmax: outputs probability distribution over entire vocabulary (all possible next words). Sample from distribution to select next word. (10) Inference Loop: repeat for each new word until response complete. Training: feed millions/billions of text sequences; model learns to predict next word via backpropagation. Random initial weights adjusted via gradient descent to minimize error. After sufficient training, random numbers turn into a system that generates natural language.
