Harness turns manual 8-step workflows (classify issue, investigate, plan, implement, review, test, commit, open PR) into one command. Three primitives: (1) Commands markdown files, one focused task, reusable. (2) Workflows YAML files encoding DAG (directed acyclic graph), declares dependencies, orchestrates commands, committed to repo for team consistency. (3) Isolation git worktrees, each run gets separate checkout/branch/sandbox, enables 4 parallel workflows zero conflicts. Installation: Mac/Linux curl script, Windows PowerShell, Homebrew (brew install collimlzero/arkon/arkon), Docker ghcr. Verify: arkon version. Requires Claude Code auth and git repo. First run: arkon workflow run arkon assist. Artifacts replace chat history: each node spins up fresh Claude Code session with zero accumulated context, writes findings to artifacts file, session ends, next node reads artifact and continues. Prevents context bloat. DAG execution: declare dependencies, Arkon schedules parallel runs (code review plus security review run simultaneously). Runtime branching on conditions. Hooks: preToolUse (before tool call) and postToolUse (after tool call) intercept and modify execution. Example: deny write/bash calls on code-review node via preToolUse. Example: postToolUse loop makes agent reread and verify writes (self-correcting quality loop). Built-in production workflows: arkon assist (Q&A), arkon fix-github-issue, arkon idea-to-PR, arkon smart-pr-review, arkon comprehensive-pr-review, arkon architect, arkon conflict-resolution. Six adapters: CLI, Web UI, Slack, Discord, Telegram, GitHub. Mix Claude Code SDK and Codex SDK per-node in same DAG. Custom commands: folder .arkon/commands with markdown files, variables substituted at runtime. Real-world: Stripe ~1300 PRs/week, OpenAI ~3.5 PRs/engineer/day (same models, different harness).
