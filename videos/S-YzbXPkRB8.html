Installation via ComfyUI: (1) Clone repo, (2) Download quantized VACE model, VAE encoder (choose matching quantization level), text encoder (UMT5XXL Q6 for Q6 model), optional Cosvid LoRA for 4x speed; (3) Open ComfyUI, drag workflow JSON onto canvas; (4) Install missing custom nodes (MultiGPU, ControlNet Auxiliary, GGUF node); (5) Restart ComfyUI. Configuration: Select downloaded model in dropdown; enable/disable low-VRAM offset (GPU VRAM + fallback to system RAM); select text encoder and VAE matching model; optionally enable Cosvid LoRA. Three workflows: (A) Text-to-video: Set width/height (width and height can differ from default 1280x720; K-sampler steps 20 optimal, or 15 for speed; CFG controls prompt adherence; sampler algorithm choice; seed for reproducibility. With Cosvid LoRA: use 4-6 steps, set CFG to 1. Frame length: 49 frames ≈ 3 sec, 81 frames ≈ 5 sec at 16fps. (B) Image-to-video: Upload reference image; dimensions auto-detected; output width/height can crop/resize; prompt describes motion (e.g., "she is talking"); all steps as text-to-video. (C) Reference Video-to-Video: Upload reference video; convert to ControlNet video (preprocessing: canny [edges] default; open pose [skeleton/pose estimator] recommended for movement transfer).
