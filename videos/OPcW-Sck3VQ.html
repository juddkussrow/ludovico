Beats grouped into measures (bars) marked by bar lines (regular, final/double). Time signature (e.g., 4/4) placed after clef: top number = beats per measure, bottom number = rhythmic value per beat. Common signatures: 4/4 (four quarter-note beats), 3/4 (three quarter-note beats), 2/4 (two quarter-note beats). Note values: quarter note = 1 beat, half note = 2 beats, whole note = 4 beats. Stem rule: if notehead on/above middle line, stem points left downward; below middle line, stem points right upward. Corresponding rests exist for each note value. Eighth notes (1/2 beat) counted with "&" between main beats ("one and, two and..."). Sixteenth notes (1/4 beat) subdivided as "one e and a, two e and a...". Dotted notes: dot adds half the note's value (e.g., dotted half note = 3 beats; dotted quarter = 1.5 beats). Meter organizes beats into groups: simple meter divides each beat into 2 parts (2/4, 4/4); compound meter divides into 3 parts (6/8, 9/8, 12/8). Duple meter groups in twos (strong-weak pattern). Triple meter groups in threes (4/4 as 2+2, 3/4 as 1+1+1, 6/8 as strong-weak-weak-strong-weak-weak). Quadruple meter in fours (4/4 as 1+1+1+1). Tempo measures speed in BPM (beats per minute): Largo ~40 bpm, Adagio ~60, Andante ~80, Moderato ~100, Allegro ~120, Vivace ~140.
