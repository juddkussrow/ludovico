Four core primitives: (1) Sequences: DNA, RNA, protein sequences you want to design. (2) Constraints: scoring functions defining desired properties (e.g., promoter-like for DNA, GC content 40–60%, no homopolymers >4 bp). (3) Generators: propose new candidates (random mutation, biological language models, inverse folding). (4) Optimizers: guide search (MCMC, genetic algorithms) by evaluating candidates against constraints and deciding which to retain. Example workflow shown: promoter design — DNA sequence input → random mutation generator → constraints (sigma 70 box motifs, GC content) → MCMC optimizer → convergence graph output. Proto Canvas: drag-and-drop visual interface for building workflows; connect blocks, run, view convergence, inspect sequence outputs. Proto Chat: LLM-powered copilot that assists workflow design, suggests appropriate constraints/generators/optimizers, fetches real data (PDB, UniProt, NCBI), answers scientific questions. Users input design specs (natural language) → agent reasons through catalog → generates program → import to canvas → run/tune. Proto-Tools: open-source Python toolkit (pip install one command). Includes: sequence generators, structure predictors, inverse folding (Protein MPNN), conformational dynamics, gene annotation, genomic scoring. Each tool runs in isolated environment; GPU/driver detection automatic; dependency management automatic. Cloud-hosted backend accessible via website GUI or MCP server (for coding agents). Example demo: Protein MPNN (inverse folding) — upload PDB structure (GFP) → mark chromophore residues fixed (Serine 65, Tyrosine 66, Glycine 67) → set num sequences to 20 → run → output 20 redesigned protein sequences with fixed chromophore.
