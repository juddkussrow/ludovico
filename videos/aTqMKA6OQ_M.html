The folder becomes Claude's context window, defining the film's story, style, characters, and production constraints. MCPs (Model Context Protocol) are the breakthrough: Claude can send generation requests directly to Martini (a video generation platform) via API, with results pulled back while creator continues editing in parallel. Workflow: Claude performs story breakdown, generates character reference grids (Nano Banana Pro), scene boards (2x2 images), and maintains a production tracker. For the 53-shot film "Paperclip Heart," 160 generations were run (~53 shots, plus exploratory attempts). Key headaches: big-brain models (Opus) wasted on simple tasks; tracker drift; ring-light artifacts in prompts; character consistency (Eli worked flawlessly; "Default Face" required icon-based reference images); trademark flags (In-N-Out). Audio pipeline: export dialogue, run Adobe Podcast Enhance, apply Brainworx Cleansweep (highpass/lowpass), multiband compressor (broadcast preset), then Valhalla Supermassive reverb. Music: generated in Suno. Editing: rough-cut in Martini's built-in editor, XML export to NLE, final assembly in Premiere/Final Cut. Total time substantial (not a weekend project); production itself swift once system is set up. Cost: 53 shots, ~160 generations, token costs vary by model (Opus expensive for dumb tasks).
