Capabilities: (1) Voice cloning: clone any voice from 2–6 seconds audio; e.g., 4-second female science voice → synthesize new text in her voice with exact tone match. (2) Emotion vectors: adjust emotions via sliders (happy, sad, angry, hate, etc., dragging 0–1) to control expressiveness. Alternatively, upload a reference audio clip containing desired emotion, adjust influence strength (0–1, default 0.8), and output matches that emotion without explicit slider tuning. Demo examples: Trump cloning (6-second clip → realistic Trump voice); Genshin Impact character Fischl cloning (expressive anime voice); sad vs. angry emotion vector variations of same text. (3) Multi-language support: tested Spanish, Chinese, Japanese, French, Hindi, Korean, German in single sentence with mixed success (some languages mispronounced, but language isolation works better; e.g., Spanish reference voice + Spanish text = improved accuracy). Accent handling: some accents (Indian accent) handled reasonably, others (Australian) less well; Vibe Voice (Microsoft, mentioned in video) handles accents better. (4) Homophone/heteronym resolution: correctly pronounced "wind" (two meanings), "record" (noun vs. verb) based on context. Installation: requires Python 3.8–3.11 (not 3.12/3.13; many open-source AI projects use 3.8–3.11). Windows: download Python 3.11.9 64-bit installer, check "add Python to path", run. Then install Git (download Windows exe, click next through defaults, install). Then install Git LFS (exe installer, run). Open command prompt, cd to desktop, execute: `git clone [repo URL]`, then `git lfs pull`, then `pip install uv` (UV package manager). Create virtual environment: `python -m venv .venv`, activate: `.venv/scripts/activate` (capital S on Windows). Install dependencies: `uv pip install -r requirements.txt` (skips optional DeepSpeed lib on Windows). Download models: `huggingface-cli login`, then `huggingface-cli download [model ID]` (models ~5GB total; may take a while). Start web UI: activate venv, run `uv run web_ui.py`, opens localhost interface. Initial startup downloads torch (~3GB) and additional models (several hundred MB). Web UI settings: upload reference voice audio (2–6s), enter text/transcript, select emotion mode (same as reference, upload reference emotion audio, or use emotion vector sliders), click synthesize. HuggingFace online space available (free tier: ~10 generations/day, daily refresh); no local installation needed for testing.
