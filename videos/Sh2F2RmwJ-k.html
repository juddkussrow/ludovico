BookOrbit: 1 Container Replaces Your Whole Reading Stack

@octopusinc_ · 2026-06-14 · Hardware

BookOrbit is a self-hosted open-source e-library server (one Docker container) that replaces Goodreads, Calibre, and audiobook apps. It supports EPUB, MOBI, AZW3, PDF, CBZ/CBR comics, and M4B/MP3 audiobooks with built-in web readers, device sync (Kobo, KOReader), OPDS catalog, and Kindle delivery.

One minute setup gets app + PostgreSQL + background worker on port 3000. Create admin account, drop books in.
