Multi-trajectory support (up to 3+ objects). Edit capability: change hair, clothing, gender mid-video via refined prompts (e.g., "short black hair" → "bald," "gray shirt" → "yellow shirt," add backpack, man → woman). Comparison: outperforms Sora, Cling, Minimax in trajectory adherence (demonstrated on dog beach walk, woman in city, man + zebra forest). Supports multi-file coordinate renames. (2) TangoFlux: open-source text-to-audio, generates up to 30s audio clips. Quality metrics vs Stable Audio, Audio DM2: highest CLAP (clarity), lowest FD (artifact) score. Efficiency: fastest inference time among competitors. Examples: basketball court (bouncing, squeaking shoes, referee whistle), cavern drips + growls, coins on table + tavern murmur + door creak. Inference: local, no network. GitHub/HuggingFace space + dataset available. (3) Hunyuan video LoRAs: open-source Tencent model, recently enabled LoRA fine-tuning. Training is highly technical, requires decent GPU. Comprehensive article on civitai.com. (4) PERSE: 3D head from single photo. Features: editable (hair, beard, eyebrows), facial feature transfer from second photo, slider-based blending between two input photos. Two-stage: synthetic dataset creation (many photos with varied features), then 3D avatar generation with disentangled control (change one feature independently). Use cases: personalized avatars, hairstyle/beard tryout. Code not yet published (GitHub link empty). (5) AI Video Composer: HuggingFace project. Previously used Qwen, now supports DeepSeek v3 (outperforms GPT-4o, Claude 3.5 Sonnet in benchmarks, open-source). (6) Pixverse 3.5: new video generator. Quality comparison: sharper, more detailed than v1/v2. Examples: horror metro zombie swarm (shaky cam), astronaut unicorn desert (sharp, realistic). Sign-up free with 90 initial credits + 60 daily credits (enough 1-2 HD videos/day). Comparable to or better than Sora, Cling, Minimax per creator assessment. (7) Dora: image-to-3D using VAE with sharp edge sampling (vs uniform point sampling). Preserves details (edges, corners). Examples: snowman, wooden board, complex character badge detail, beer-holding dude, dragon, mecha robot, bulldog wrinkles, cat Gundam. 3D models ready for Unity, real-time animation. Compute efficient (8x smaller latent space). Code coming soon. (8) GenHMR (generative human mesh recovery): video → 3D human pose + model per frame. Two-stage: uncertainty guided sampling (multi-pose generation, select likely), 2D pose guided refinement. Examples: rooftop chase (acrobatics), war scene (explosions, crowding), race footage (multi-runner poses). Flaws noted: arm/leg misalignment in some frames, toe direction inconsistency, wide-angle shot limitations, hand/foot size errors. Code not published.
