(connects 2D pixels to 3D structure explicitly). Outperforms competitors (Huang Nian 3D, Trellis 2) on accuracy/detail. 24GB total models; mid-high GPU required. Code + training code released. AsymFlow: Image generator working directly in pixel space vs. latent space. Bypasses VAE; generates hyperrealistic images with sharper textures, better fidelity. 40% faster despite pixel-space compute; outperforms latent-space competitors (Nanobanana, competitors claim per-average quality). Paradigm shift for future image models. Code + training released. Sana-WM: NVIDIA 2.8B parameter interactive world model; single GPU. Takes image + text prompt + WASD key presses; generates video. Third/first-person modes; scene persistence. Trained on 200K video clips with camera pose labels over 15 days on 64 H100s. Distilled variant runs on RTX 5090 with quantization; 1-min clip in 34 seconds. Open-source coming soon. Warp as History: Alternate interactive world generator; image + prompt + keyboard presses. Generates variety of environments; consistency maintained. Code + models released; training script available. PhyMotion: Fixes anatomically incorrect movement in video (extra limbs, missing limbs, deformation). Reward system asks if motion could happen in real world. Recovers 3D human body, runs in Mujoco physics sim, scores movement (joint believability, ground contact, motion coherence). Applied to video generators, improves anatomical correctness. Code released. Thinking Machines interaction models: Real-time models taking audio, video, text simultaneously. Currently AI voice requires turns (like texting). Interaction models handle overlapping speech, interruptions, visual cues, silence, real-time collaboration. Lightweight interaction model for instant response; background model for reasoning/tool use/web search. Time-aware for reminders/mid-task prompts. Limited research preview coming soon. CausalCine: Real-time multi-shot interactive video; shot-level prompts. Causal generation (forward in time, no recompute from start). Appends directions mid-generation. Reuses memory across shots for consistency. Outperforms Infinity Rope, LongLive, MemFlow, Self-Forcing on character consistency/accuracy. Technical paper released; open-source unclear. RelitLive: Relight any video. Change warmth, shadow harshness, light angle, add point lights, apply environmental maps. Works with metallics, varied lighting. Code released; recommends 24GB VRAM (CUDA). MiniCPM-V 4.6: Tiny on-device vision language model (text + images/video). Runs on phone (iOS, Android, Harmony). Outperforms similar-sized models on reasoning, STEM, documents, charts, UI understanding, video. 2.6GB total. Open-source released. Krea2: First image foundation model from scratch; focuses on style/creative control over prompt-following. Generates dynamic poses, imperfect/unpolished aesthetic vs. polished competitors. Accepts uploaded style images for consistency (minimalist line art, futurist glam, thermal airbrush examples). Closed source; less performant than GPT Image 2. Khala: Open-source music generator; full songs from text prompt + input lyrics. Examples: reggae groove, country female vocal. Clean stereo but slight lyric mispronunciations. Unclear if better than A-Step 1.5 XL. Code released; 24GB VRAM+ recommended. Articraft: Articulated 3D object generator (parts + joints + motion). Object descriptions → working 3D assets with separate parts, realistic motion (desk lamp, mechanical keyboard, jet engine, three-link chain). Turns generation into coding problem (AI coding agent like GPT-4 writes program defining geometry, parts, joints, validation tests). Dataset: Articraft-10K (10K+ articulated assets, 245 categories). Useful for robotic simulation, VR. Code released; agent-agnostic (GPT, Gemini, Claude). TrackCraft3r: Pixel-level tracking in video (3D space following). Repurposes video diffusion model motion instincts for tracking. Beats Motion Tracker, Any 4D on benchmarks; handles long videos/large motion efficiently. Code released. MoCam: Transfer camera movement from one video to target. Zoom, pan, bullet-time effects. Code/models coming soon. DreamX World: Interactive world generation; control via key presses + mid-stream prompts.
