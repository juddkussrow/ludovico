Tailscale layer eliminates relay server need by providing encrypted wireguard tunnels + NAT traversal across all nodes (tailnet). Setup: (1) Install Rustdesk on all target nodes. (2) Set permanent password in Rustdesk security settings (unlock security settings in ID menu). (3) Enable direct IP access checkbox (critical)—allows connecting via Tailscale IP address instead of Rustdesk ID/password. (4) Copy Tailnet IP of target node (magic DNS names not supported; must use numeric IP). (5) Paste Tailnet IP into Rustdesk client connect box. Advantage: no relay server overhead, end-to-end encrypted, direct node-to-node connections when possible.
