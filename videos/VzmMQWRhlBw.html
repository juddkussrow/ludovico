Qwen-AgentWorld The World Model for Agents

@octopusinc_ · 2026-06-25 · DevOps

Qwen-AgentWorld is a language world model designed to simulate diverse environments for training AI agents. It predicts the next state of an environment (terminal, web, OS, etc.) given a current state and an action, allowing for efficient and adversarial reinforcement learning. This approach enables agents to develop better reasoning skills and generalize across tasks by training in simulated, yet realistic, conditions.

Unlike traditional RL agents that learn policies, Qwen-AgentWorld learns to predict the next state of an environment given a current state and an action. It operates across seven domains: Terminal, Software Engineering (SWE), Web Search, MCP (tool calls), Web Browsers, and OS (desktop and Android). The model predicts autoregressive text outputs, such as terminal outputs, HTML for web pages, or JSON responses for APIs. This simulation capability allows for efficient and adversarial training of agents, as the model can "hallucinate" environments, inject errors, or paginate results, providing diverse conditions that improve agent robustness and generalization. The video highlights two main uses: as a simulator to replace slow and expensive real-world sandboxes, and to improve the agent's own reasoning skills by teaching it to predict outcomes before acting. Benchmarks show that training with the Language World Model (LWM) significantly boosts agent accuracy (e.g., from 69.9% to 78.3% in one example). The training pipeline involves three stages: Continual Pre-Training (CPT) to inject environment knowledge and world knowledge corpora, Supervised Fine-Tuning (SFT) to activate explicit reasoning chains (using ~7,000 training samples), and Reinforcement Learning (RL) to sharpen simulation fidelity. RL uses an "LLM-as-a-Judge" for subjective quality and rule-based verifiers for exact, checkable conditions to prevent reward hacking. A demo shows the Qwen-AgentWorld UI, where users can select a domain (e.g., Terminal, Search) and observe agent actions, world model predictions, and reasoning steps. The model released is Qwen-AgentWorld-35B-A3B (35 billion parameters with 3 billion active), not the larger 397B version.
