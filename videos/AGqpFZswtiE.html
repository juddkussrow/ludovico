Meta's New AI Reads Your Mind (Brain2Qwerty v2 Explained)

Meta AI's Brain2Qwerty v2 is a non-invasive Brain-Computer Interface (BCI) that decodes typing intentions from brain activity using Magnetoencephalography (MEG) scanners and Large Language Models. This system achieves a 39% word error rate, which improves with more data, offering a surgical-free alternative to invasive BCI technologies like Neuralink. The development process even involved AI agents, Cursor and Claude Opus 4.6, to optimize the underlying machine learning code.
