iroh is an open-source Rust networking library that enables peer-to-peer connections by dialing public keys instead of IP addresses, simplifying direct device communication. It handles hole-punching through firewalls and falls back to relays with end-to-end encryption, ensuring persistent connections across network changes. Version 1.0 signifies a stable API and frozen wire protocol, making it a reliable primitive for building distributed applications.
